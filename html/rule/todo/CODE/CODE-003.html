CODE-003: dog: collapse table-scan & twin helper dups

dog repeats several small structural blocks: dog_proj_lookup/DOGIsTransport/DOGIsGitTransport walk a string table with the same loop, the ron60→time_t conversion is pasted across HUNK.c, WHIFF's 40/60 hashlet helpers are byte-twins on one shift constant, and ULOGVerbColor/ULOGVerbTag scan one table identically. Factor each into a single parameterized helper.

Issues

Blockers

None; dog-internal. Shares DOG.c/HUNK.c with CODE-002, so land in the same dog worktree, serially.
