CODE-012: woof/CONN.c: abc unescape/escape, slice slots

woof/CONN.c hand-rolls percent-decoding (= URIu8sUnesc), copies dog/HUNK.c's html-escape verbatim, and manages per-conn buffers with raw pointer arithmetic over WOOF.pool. It also duplicates the HTTP-200+HTML-prelude emit and the free-slot reset. Route through abc/URI and a shared html-escape, and carve slots with abc slices.
