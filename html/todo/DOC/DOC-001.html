DOC-001: rework all INDEX.md to the meta/index.mkd format

Rework every production-module-dir INDEX.md to match index: a top intro, then ONE SECTION PER .h file — each a short mission intro + a ≤12-bullet list of the module's KEY types/functions (one-liner each), sections grouped by layer/theme. Explanatory tables/drawings move OUT of the INDEX into per-file .md docs (e.g. abc/S.md), linked from the section intro; those .md docs are created/expanded as part of this work. Docs-only, but every entry must be verified against the real header. Method: Issues.
