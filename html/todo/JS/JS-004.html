JS-004 HIGH: len-1 / factlen-1 size_t underflow on empty / zero-length strings

JSStringGetUTF8CString returns 0 on an empty string or failure; the -1 sites then underflow size_t to a huge value. Flagged by all 7 lenses (highest confidence). Method: Issues.
