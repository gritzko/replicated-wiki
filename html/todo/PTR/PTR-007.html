PTR-007: bro/+woof/ raw pool arena, manual fragment parse

The 2026-06-15 audit found woof/ backs its connection pool with a raw u8 * from mmap, forcing per-slot pointer arithmetic and four-cell buffer writes throughout CONN.c; bro/ hand-parses the prompt/URI path with char[]+strlen/atoi/sscanf. Behavior-preserving; *woof is a load-bearing spawn/serve path — gate the diff before landing.*

Input

Context — woof/

  • woof/WOOF.c:62-65 (+ woof/WOOF.h:146) — WOOF.pool is a raw u8* from mmap, so WOOFConnCarve computes base=pool+slot*SLOT_BYTES, p_pin=p_in+REQBUF, p_out=p_pin+PIPEBUF and writes all four cells of c->in/pipe_in/out by hand; slot_release/serve-inproc repeat pool+slot*SLOT+off (CONN.c:125-953). ROOT CAUSE.

Goals

  • Make WOOF.pool a u8a/u8b mapped arena; carve slots and sub-regions with u8bAcquire/u8bMap slice helpers — collapses the ~6 pool+slot*SLOT+off sites and the four-cell c->in/pipe_in/out writes.
  • woof: LIT→existing u8slit (delete the macro); cell-pokes→u8bUsed/u8bIdle; Fork-by-hand→u8sFork/u8sJoin (or feed c->out directly); envelopes→u8csc; {argv_a,argv_a+3}→a$(u8cs, argv, argv_a); au.data restore→$mv/u8csMv.
  • bro: route the fragment/URI path through u8csc/path8sc + ragel URILexer + typed trims; completion → a u8b of interned word slices iterated with $for; bro_lineinfo → a typed Bbro_lineinfo; scr_emit_char→u8csc chars; two-head slice→$mv(text, u8bDataC(mapped)); MAUS→u8csc.

Constraints

  • woof is load-bearing — gate the diff before landing; behavior-preserving; ctest -j16 --timeout 10 green.
  • Int-fd tables (i32b maps, conn sock_fd/pipe_fd) are real POSIX fds — not slice candidates.
