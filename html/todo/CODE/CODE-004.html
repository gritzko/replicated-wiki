CODE-004: keeper: one commit-parents/BFS/sha40 helper

keeper open-codes commit ancestry in many places: three near-identical BFS loops in KEEP.c (IsAncestor/SharesAncestor), the 40-hex→sha1 decode at 7+ sites (the file-static keep_hex40_sha in WALK.c is unused elsewhere), and WIRECLI's own parent-walks. Extract a commit-parents collector, a bounded BFS-with-seen, and a shared sha40 decoder.

None; keeper-internal. keep_hex40_sha already exists in WALK.c but is file-static and unused elsewhere.
