CODE-019: relocate sha1hex from dog/WHIFF.h to dog/git/SHA1.h (beside its sibling sha1)

dog/WHIFF.h is about packed index words (wh64/wh128) + SHA→hashlet keys — but it ALSO defines sha1hex (the 40-ASCII-hex SHA-1 struct) and its bin↔hex conversions, which WHIFF's own logic never uses. Its sibling, the binary sha1 struct + SHA-1 hashing, already lives in dog/git/SHA1.h (which even has a 40-hex feed helper). sha1hex belongs there. TIGHT, behaviour-preserving move + include fixes — NOT an open cleanup. Method: Issues.
